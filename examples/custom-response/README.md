## custom response writer

default response writer

```
{
  "name": "string",
  "nickName": "string",
  "phone": "string"
}
```


custom response writer `umi.RespWriter`

```
{
  "data": {
    "name": "333",
    "nickName": null,
    "phone": "12345"
  },
  "host": "yubo-didi-mbp16",
  "errorCode": "string",
  "errorMessage": "string",
  "success": true
}
```
